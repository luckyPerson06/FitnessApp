# Fitness App (Grain)

[![SonarQube](https://img.shields.io/badge/SonarQube-Анализ%20кода-blue?logo=sonarqube)](https://sonarcloud.io/summary/overall?id=luckyPerson06_FitnessApp&branch=main)

## 📋 О проекте
Backend-приложение для управления фитнес-клубом. Реализует полный цикл работы с клиентами, тренерами, абонементами и расписанием тренировок.

## 🏗️ Функциональность

### 👥 Управление клиентами
- Регистрация и управление профилями
- Привязка абонементов
- История посещений

### 👨‍🏫 Управление тренерами
- Профили и специализация
- Расписание тренировок
- Назначение типов тренировок

### 📅 Расписание
- Создание тренировочных сессий
- Проверка пересечений у тренеров
- Отслеживание свободных мест

### 🎟️ Абонементы
- Различные типы (LIMITED/UNLIMITED)
- Привязка к типам тренировок
- Контроль использования

### ✅ Посещения
- Запись на тренировки
- Отметка посещений
- Статистика по часам

## 🛠️ Технические особенности

### Архитектура
- Java 17 + Spring Boot 3.3.5
- JPA/Hibernate с PostgreSQL
- MapStruct для маппинга DTO
- Flyway для миграций

### Модель данных
- 6 связанных сущностей
- Связи OneToMany и ManyToMany
- Индексы для оптимизации запросов
- Версионирование через @Version

### Демонстрационные возможности
- Проблема N+1 и её решение через JOIN FETCH
- Частичное сохранение (без @Transactional)
- Атомарные операции с rollback (с @Transactional)
- Проверка пересечений расписания
